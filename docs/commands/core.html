bl add

  1. Add a user to this server's blacklist, preventing the user from calling commands.
    • <user>
  2. Add a user to the global blacklist. Owner only. `--global` or `-g` specify global
    • <user>
    • <--global>
    • <-g>

bl remove

  1. Remove a user from this server's blacklist.
    • <user>
  2. Remove a user from the global blacklist. Owner only. `--global` or `-g` specify global
    • <user>
    • <--global>
    • <-g>

bl ls

  1. List users in the blacklist
